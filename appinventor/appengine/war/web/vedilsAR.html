VEDILS Augmented Reality

La tecnología de realidad aumentada permite superponer elementos virtuales sobre el mundo real en tiempo real. Para poder integrar esta tecnología en nuestros dispositivos móviles VEDILS pone a nuestra disposición los elementos necesarios. Los elementos y su relación se muestra en la siguiente imagen.

Elementos para implementar realidad aumentada
  • ARCamera
    • Es el componente que lanza la cámara para empezar a reconocer nuestro entorno.
    • Pueden definirse parámetros como posición del móvil.
    • Cuando utilizamos un ARObjectTracker, debemos definir dos parametros: fichero .DAT y fichero .XML. Estos son la báse de datos obtenidas del portal de Vuforia con las imágenes que queremos reconocer.
  • Tracker
    • Son los elementos que queremos reconocer.
    • El componente ARTextTracker nos permite reconocer texto.
    • El componente ARMarkerTracker es capaz de reconocer hasta 512 marcas.
    • Para reconocer imágenes utilizamos ARObjectTracker que deberemos configurar con un nombre de la base de datos configurado en la cámara con los ficheros .DAT y .XMl de Vuforia
  • Asset
    • Elementos virtuales que se visualizarán.
    • El componente ARImageAsset nos permite mostrar imágenes 2D.
    • Con el componente AR3DModelAsset podemos poner imágenes 3D en formato .OBJ, .3DS y animaciones .MD2.
    • Este tipo de elemento debe relacionarse con una cámara y un Asset
  • ARCameraOverLayer
    • Sirve para poner información sobre la imagen capturada por la cámara en tiempo real.
    • Esta preparado para poner texto y un par de botones.

Reconocimiento de marcas

VEDILS incoropora las marcas de Vuforia, pudiendo reconocer cualquiera de sus 512 marcas, para ello debemos, en la ventana de diseño, especificar a que número de marca corresponde el componente ARMarkerTracker. El uso de este componente se puede ver en el siguiente video.




Reconocimiento de imágenes

VEDILS puede reconocer imágenes de nuestro entorno, para eso debemos utilizar la herramienta de para crear la base de datos de imágenes. Esta base de datos cuando la descarguemos obtendremos un fichero .ZIP con dos ficheros, uno .XML y otro .DAT que habrá que subir a VEDILS y configurar en el componenete ARCamera. Luego en el componente ARObjecTracker deberemos indicar el nombre de la imagen de la base de datos creada que deseamos utilizar. El uso de este componente se puede ver en el siguiente video.



Reconocimiento de texto

VEDILSno permite reconocer palabras, para ello solo deberemos configurar en el componente ARTextTracker la palabra que queremos que se reconozca. El uso de este componente se puede ver en el siguiente video.
